# Hermes

Prototipo de sistema de comunicación asistida basado en bioseñales.

---

## 🧠 Descripción
Hermes es un proyecto en desarrollo enfocado en la creación de una interfaz de comunicación asistida utilizando señales electromiográficas (EMG), principalmente de la región facial.

El objetivo es explorar la traducción de actividad muscular en texto o comandos, permitiendo a usuarios con limitaciones del habla interactuar con su entorno de forma autónoma.

Actualmente, el proyecto se encuentra en una etapa temprana de desarrollo.

---

## ⚙️ Estado del proyecto
🚧 En desarrollo activo

Este repositorio contiene implementaciones iniciales, pruebas de concepto y experimentación con:
- Adquisición de señales EMG  
- Procesamiento de señales  
- Modelos de interpretación  
- Interfaces básicas de salida  

Nada aquí es definitivo. Todo está sujeto a cambios.

---

## 🎯 Objetivo
Desarrollar una base tecnológica para sistemas de comunicación asistida accesibles, no invasivos y adaptables a distintos usuarios.

---

## 🧩 Componentes (tentativo)
- `signal/` → adquisición y manejo de señales  
- `processing/` → filtrado y extracción de características  
- `models/` → interpretación de señales  
- `interface/` → salida (texto/comandos)  

*(estructura sujeta a cambios conforme evolucione el proyecto)*
